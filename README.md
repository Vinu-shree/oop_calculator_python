# OOP Calculator in Python

This project is a simple calculator built using Object Oriented Programming (OOP) concepts in Python.

## Features
- Addition
- Subtraction
- Multiplication
- Division
- Continuous operation using loop
- Handles invalid operator input
- Prevents division by zero

## Concepts Used
- Classes and Objects
- Constructor (__init__)
- Methods / Functions
- Conditional Statements
- While Loop
- User Input Handling
